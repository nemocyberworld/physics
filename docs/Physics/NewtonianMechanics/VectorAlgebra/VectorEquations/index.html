Physics Topic: Vector Equations

1. Vector Equation of a Line

The vector equation of a line in space can be expressed in the form:

$$ \mathbf{r} = \mathbf{a} + t\mathbf{b} $$

Where:

  • r is the position vector of any point on the line.
  • a is the position vector of a known point on the line.
  • b is a direction vector along the line.
  • t is a scalar parameter that can vary to describe all points on the line.

For example, if we know that point A has coordinates (1, 2, 3) and the direction vector of the line is b = (4, 5, 6), the vector equation of the line passing through A is:

$$ \mathbf{r}(t) = (1, 2, 3) + t(4, 5, 6) $$

As t changes, we get different points along the line.

2. Vector Equation of a Plane

The vector equation of a plane can be written as:

$$ \mathbf{r} = \mathbf{a} + s\mathbf{b} + t\mathbf{c} $$

Where:

  • r is the position vector of any point on the plane.
  • a is the position vector of a known point on the plane.
  • b and c are two non-parallel vectors lying on the plane.
  • s and t are scalar parameters that can vary to describe all points on the plane.

For example, if point A has coordinates (1, 2, 3), vector b = (4, 0, 0), and vector c = (0, 3, 0), the vector equation of the plane becomes:

$$ \mathbf{r}(s, t) = (1, 2, 3) + s(4, 0, 0) + t(0, 3, 0) $$

This equation describes all points on the plane passing through point A.

3. Shortest Distance Between Two Lines

The shortest distance between two skew lines (lines that do not intersect and are not parallel) can be calculated using the formula:

$$ d = \frac{|\mathbf{(r_2 - r_1)} \cdot (\mathbf{b_1} \times \mathbf{b_2})|}{|\mathbf{b_1} \times \mathbf{b_2}|} $$

Where:

  • r1 and r2 are position vectors of points on lines 1 and 2 respectively.
  • b1 and b2 are direction vectors of lines 1 and 2.
  • • represents the dot product, and Ã— represents the cross product.

For example, consider the lines with position vectors:

  • r1 = (1, 1, 1) and direction vector b1 = (1, 0, 0).
  • r2 = (0, 2, 1) and direction vector b2 = (0, 1, 0).

Using the formula, we can compute the shortest distance between these two skew lines.

4. Coplanarity of Vectors

Vectors are said to be coplanar if they lie on the same plane. The condition for coplanarity of three vectors a, b, and c is:

$$ \mathbf{a} \cdot (\mathbf{b} \times \mathbf{c}) = 0 $$

If the scalar triple product equals zero, the vectors are coplanar. Otherwise, they are not coplanar.

For example, consider the vectors:

  • a = (1, 2, 3), b = (4, 5, 6), and c = (7, 8, 9).

To check if these vectors are coplanar, we compute the scalar triple product:

$$ \mathbf{a} \cdot (\mathbf{b} \times \mathbf{c}) = (1, 2, 3) \cdot ((4, 5, 6) \times (7, 8, 9)) $$

If the result is zero, the vectors are coplanar. Otherwise, they are not.

Practical Applications

  • Vector equations are essential in physics, particularly in mechanics and electromagnetism, where the motion of particles and fields can be described using vectors.
  • In engineering, the concept of vector equations helps in structural analysis, especially when working with forces acting on objects in space.
  • In computer graphics, vectors are used to represent objects, light, and motion in 3D space.

5. Python Example

import numpy as np

# Defining the vectors
r1 = np.array([1, 1, 1])
b1 = np.array([1, 0, 0])
r2 = np.array([0, 2, 1])
b2 = np.array([0, 1, 0])

# Calculating the shortest distance between two skew lines
cross_product = np.cross(b1, b2)
numerator = np.abs(np.dot(r2 - r1, cross_product))
denominator = np.linalg.norm(cross_product)
distance = numerator / denominator
print("Shortest distance:", distance)

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from mpl_toolkits.mplot3d import Axes3D

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Parameters
t_vals = np.linspace(0, 10, 200)

# 1. Straight Line: r = a + tb
a = np.array([0, 0, 0])
b = np.array([1, 2, 1])
line_points = np.array([a + t * b for t in t_vals])

# 2. Plane: r = p + su + tv
p = np.array([0, 0, 0])
u = np.array([1, 0, 0])
v = np.array([0, 1, 1])
s_vals = np.linspace(-2, 2, 10)
t_vals_plane = np.linspace(-2, 2, 10)
S, T = np.meshgrid(s_vals, t_vals_plane)
plane_points = p[:, None, None] + S * u[:, None, None] + T * v[:, None, None]

# 3. Helix: r(t) = (cos(t), sin(t), t)
helix_points = np.array([[np.cos(t), np.sin(t), t] for t in t_vals])

def update(frame):
    ax.cla()
    ax.set_title("Vector Equation Animations")
    ax.set_xlim(-5, 5)
    ax.set_ylim(-5, 5)
    ax.set_zlim(-2, 12)
    ax.set_xlabel("X")
    ax.set_ylabel("Y")
    ax.set_zlabel("Z")

    # Draw Line Path
    ax.plot(line_points[:,0], line_points[:,1], line_points[:,2], color='blue', label="Line: a + tb")
    ax.quiver(*a, *(line_points[frame] - a), color='blue', length=1, normalize=True)

    # Draw Plane
    ax.plot_surface(plane_points[0], plane_points[1], plane_points[2], color='orange', alpha=0.4, edgecolor='none')
    ax.text(*p, "Plane Origin", color='black')

    # Draw Helix
    ax.plot(helix_points[:frame,0], helix_points[:frame,1], helix_points[:frame,2], color='purple', label="Helix: (cos t, sin t, t)")
    ax.quiver(0, 0, 0, *helix_points[frame], color='purple', linewidth=2)

    ax.legend()

ani = FuncAnimation(fig, update, frames=len(t_vals), interval=50)
plt.show()
