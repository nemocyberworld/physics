Physics Topic: Fundamentals of Vectors

1. Scalars vs Vectors

In physics, quantities are broadly classified into two categories:

  • Scalars - Quantities that have only magnitude (size). Examples: Mass, Temperature, Time, Speed.
  • Vectors - Quantities that have both magnitude and direction. Examples: Displacement, Velocity, Force.

Example: 5 kg (scalar), 5 m North (vector)

2. Geometric Representation of Vectors

Vectors are usually represented as arrows. The length of the arrow represents the magnitude, and the direction of the arrow represents the direction of the vector.

For example, an arrow 5 cm long pointing east may represent a velocity of 5 m/s towards the east.

\[ \vec{A} = 5 \text{ m/s (East)} \]

3. Notation and Conventions

Vectors are often denoted with an arrow over the symbol (\( \vec{A} \)), boldface ( A ), or both in textbooks.

  • \( \vec{A} \) denotes vector A
  • |\( \vec{A} \)| or simply A denotes the magnitude of vector A
  • Vectors follow head-to-tail addition

4. Types of Vectors

a. Zero Vector

A vector with zero magnitude and no direction. Denoted as \( \vec{0} \).

\[ \vec{0} = 0\hat{i} + 0\hat{j} + 0\hat{k} \]

b. Unit Vector

A vector with a magnitude of one unit. Used to specify direction.

\[ \hat{a} = \frac{\vec{A}}{|\vec{A}|} \]

Standard unit vectors:

  • \( \hat{i} \) along x-axis
  • \( \hat{j} \) along y-axis
  • \( \hat{k} \) along z-axis

c. Position Vector

A vector that represents the position of a point in space relative to an origin.

\[ \vec{r} = x\hat{i} + y\hat{j} + z\hat{k} \]

d. Equal and Negative Vectors

  • Equal Vectors: Two vectors are equal if they have the same magnitude and direction.
  • Negative Vectors: A vector with the same magnitude but opposite direction.
\[ \vec{A} = -\vec{B} \Rightarrow \text{Same magnitude, opposite direction} \]

e. Collinear and Coplanar Vectors

  • Collinear Vectors: Vectors lying along the same line or parallel lines.
  • Coplanar Vectors: Vectors lying in the same plane.

Example: Two forces acting on an object in the same direction are collinear; three velocity vectors acting in the x-y plane are coplanar.

f. Reciprocal Vectors

If two non-zero vectors \( \vec{A} \) and \( \vec{B} \) are such that:

\[ \vec{A} \cdot \vec{B} = 1 \]

Then \( \vec{B} \) is called the reciprocal vector of \( \vec{A} \).

5. Python Examples

# Representing and analyzing vectors using NumPy, with colors and explanations
    import numpy as np
    from colorama import Fore, Style, init
    
    init(autoreset=True)  # Automatically reset color after each print
    
    print(Fore.CYAN + "=== Vector Fundamentals with NumPy ===\n")
    
    # 1. Zero Vector
    print(Fore.YELLOW + "1. Zero Vector:")
    print("A zero vector has no magnitude or direction. It is the additive identity for vectors.")
    zero_vector = np.array([0, 0, 0])
    print("   Vector:", Fore.GREEN + str(zero_vector))
    print("   Magnitude:", Fore.MAGENTA + str(np.linalg.norm(zero_vector)) + "\n")
    
    # 2. Unit Vectors
    print(Fore.YELLOW + "2. Unit Vectors:")
    print("Unit vectors have magnitude 1 and indicate direction along coordinate axes.")
    unit_x = np.array([1, 0, 0])
    unit_y = np.array([0, 1, 0])
    unit_z = np.array([0, 0, 1])
    print("   X-axis:", Fore.BLUE + str(unit_x))
    print("   Y-axis:", Fore.BLUE + str(unit_y))
    print("   Z-axis:", Fore.BLUE + str(unit_z) + "\n")
    
    # 3. Position Vector
    print(Fore.YELLOW + "3. Position Vector:")
    print("A position vector locates a point in space relative to the origin.")
    position_vector = np.array([3, 4, 0])
    magnitude_pos = np.linalg.norm(position_vector)
    print("   Vector:", Fore.GREEN + str(position_vector))
    print("   Magnitude (length):", Fore.MAGENTA + str(round(magnitude_pos, 2)))  # Expected 5.0 (3-4-5 triangle)
    direction_cosines = np.round(position_vector / magnitude_pos, 2)
    print("   Direction cosines:", Fore.CYAN + str(direction_cosines) + "\n")
    
    # 4. Reciprocal Vectors
    print(Fore.YELLOW + "4. Reciprocal Vectors:")
    print("Two vectors A and B are reciprocal if their dot product is 1: A · B = 1")
    A = np.array([2, 0, 0])
    B = np.array([0.5, 0, 0])
    dot_AB = np.dot(A, B)
    print("   Vector A:", Fore.GREEN + str(A))
    print("   Vector B:", Fore.GREEN + str(B))
    print("   A · B =", Fore.MAGENTA + str(dot_AB))
    
    if np.isclose(dot_AB, 1.0):
        print(Fore.GREEN + "   ✅ A and B are reciprocal vectors.\n")
    else:
        print(Fore.RED + "   ❌ A and B are not reciprocal.\n")
