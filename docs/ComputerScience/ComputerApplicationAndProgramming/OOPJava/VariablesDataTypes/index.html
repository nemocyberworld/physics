Java OOP: Variables and Data Types

Introduction

Variables and data types are the foundation of any programming language, including Java. In Object-Oriented Programming (OOP), variables store the state of an object. Understanding how to use them is essential to writing good Java code.

Fun Fact!

The word "variable" comes from Latin "variabilis," meaning changeable — and that’s exactly what variables are: named containers for data that can change!

1. What is a Variable?

A variable is a name that refers to a memory location where data is stored. In Java, every variable must have a type — which determines what kind of data it holds.

2. Java Variable Syntax

type variableName = value;

Example:

int age = 25;
String name = "Alice";
boolean isJavaFun = true;

3. Types of Variables

  • Local variables: Declared inside methods
  • Instance variables: Belong to objects
  • Class variables (static): Shared across all objects

4. Java Data Types

Primitive Data Types:

  • int – whole numbers (e.g., 10, -5)
  • double – decimal numbers (e.g., 3.14)
  • boolean – true or false
  • char – single character (e.g., 'A')

Non-Primitive Data Types:

  • String – sequence of characters (e.g., "Hello")
  • Arrays, Classes, Interfaces etc.
