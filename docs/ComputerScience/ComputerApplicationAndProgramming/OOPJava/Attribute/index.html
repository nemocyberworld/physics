Java OOP: Attributes and Behavior

1. Introduction

In Object-Oriented Programming (OOP), every object has two main components: attributes and behavior. Attributes are the data (also called fields or properties), and behavior is what the object can do (methods).

2. Attributes (Fields)

Attributes describe the state or characteristics of an object. Think of them as variables inside a class.

class Dog {
  String name;
  int age;
  String breed;
}

In the above example, name, age, and breed are attributes of a Dog.

3. Behavior (Methods)

Behavior defines what an object can do. It is represented by methods in a class.

class Dog {
  void bark() {
    System.out.println("Woof!");
  }
  void eat() {
    System.out.println("The dog is eating.");
  }
}

4. Real-World Examples

  • Car: Attributes: color, speed, model | Behavior: drive(), brake()
  • Phone: Attributes: brand, OS, screenSize | Behavior: call(), text(), browseInternet()
  • Book: Attributes: title, author, pages | Behavior: open(), read()

5. Why Are They Important?

Attributes and behavior help encapsulate data and logic within objects. This leads to more modular, reusable, and maintainable code. For example, a Student class can hold student data and behaviors like attendClass() or submitAssignment().

6. Fun Fact!

In Java, every method is tied to a class‚Äîeven the main() method! Java treats everything as a class-based object model, unlike some languages that allow standalone functions.

7. Practical Applications

Attributes and behavior are used in:

  • Game Development: Characters with health, strength, attack() behavior
  • Banking Systems: Accounts with balance, withdraw() behavior
  • Shopping Apps: Products with price, addToCart() behavior

8. Creating and Using a Class

class Student {
  String name;
  int grade;

  void study() {
    System.out.println(name + " is studying.");
  }
}

public class Main {
  public static void main(String[] args) {
    Student s1 = new Student();
    s1.name = "Alice";
    s1.grade = 10;
    s1.study();
  }
}

9. Recap

  • Attributes are variables inside a class (state).
  • Behavior are methods that describe what an object does (actions).
