Java Programming: Expressions and Operators

Expressions are constructs made up of variables, literals, method calls, and operators that evaluate to a single value. Operators are symbols that perform operations on variables and values.

Fun Fact: The first programming languages didn't have many of the operators we use today—developers had to write long functions just to compare values!

Types of Java Operators

  • Arithmetic Operators
  • Relational (Comparison) Operators
  • Logical Operators
  • Assignment Operators
  • Unary Operators
  • Bitwise Operators
  • Ternary Operator
  • Instanceof Operator

1. Arithmetic Operators

Used for mathematical operations:

int a = 10;
int b = 5;
int sum = a + b;      // 15
int diff = a - b;     // 5
int prod = a * b;     // 50
int div = a / b;      // 2
int mod = a % b;      // 0

2. Relational Operators

Used to compare two values. Result is boolean.

System.out.println(a > b);  // true
System.out.println(a == b); // false

3. Logical Operators

Used to combine multiple boolean expressions:

boolean x = true;
boolean y = false;
System.out.println(x && y); // false
System.out.println(x || y); // true
System.out.println(!x);     // false

4. Assignment Operators

Used to assign values to variables.

int number = 10;
number += 5;  // number = number + 5 → 15

5. Unary Operators

Operate on one operand.

int a = 5;
int b = -a;    // -5
int c = ++a;   // pre-increment
int d = a--;   // post-decrement

6. Bitwise Operators

Operate on individual bits:

int a = 5;  // 0101
int b = 3;  // 0011
System.out.println(a & b); // 0001 → 1

7. Ternary Operator

Short form for if-else condition:

int max = (a > b) ? a : b;

8. instanceof Operator

Checks whether an object is of a specific type:

String str = "Hello";
System.out.println(str instanceof String);  // true

Real-World Examples

Banking System

double balance = 1000.0;
balance -= 200.0;  // Withdrawal
balance += 500.0;  // Deposit

Login System

boolean isUser = true;
boolean hasPassword = true;
boolean canLogin = isUser && hasPassword;

Grade Evaluation

int score = 85;
String result = (score >= 60) ? "Pass" : "Fail";

Shopping Cart

int items = 3;
double price = 9.99;
double total = items * price;
boolean eligibleForDiscount = total > 25;

Temperature Check

double temp = 37.5;
boolean hasFever = temp > 37.0;
