Java Programming: Literals

Literals in Java are the constant values that are assigned to variables. These values are fixed and do not change during program execution.

Fun Fact: Did you know? The word "literal" comes from Latin "littera," meaning "letter." In programming, it refers to exactly what is written—no interpretation needed!

Types of Literals in Java

  • Integer Literals
  • Floating-point Literals
  • Character Literals
  • String Literals
  • Boolean Literals
  • Null Literal

1. Integer Literals

These are whole numbers like 100, -25. They can be expressed in decimal, octal, hexadecimal, and binary form.

int a = 100;          // Decimal
int b = 0123;         // Octal (starts with 0)
int c = 0x7B;         // Hexadecimal (starts with 0x)
int d = 0b1010;       // Binary (starts with 0b)

2. Floating-point Literals

Numbers with decimals. Default type is double.

float pi = 3.14f;
double gravity = 9.8;

3. Character Literals

Single characters enclosed in single quotes.

char letter = 'A';
char digit = '5';

4. String Literals

Text enclosed in double quotes.

String greeting = "Hello, World!";

5. Boolean Literals

Only two values: true and false.

boolean isJavaFun = true;

6. Null Literal

Used to indicate no value or object reference.

String name = null;

Real-World Example

Consider a Student management system:

String studentName = "Alice";
int age = 20;
boolean isEnrolled = true;
double GPA = 3.75;

Examples in Context

// Bank account
String accountHolder = "John Doe";
int accountNumber = 12345678;
double balance = 1000.75;
boolean isActive = true;
// Weather report
String day = "Monday";
double temperature = 29.4;
boolean isRaining = false;
