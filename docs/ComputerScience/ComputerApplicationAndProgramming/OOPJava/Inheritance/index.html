Java OOP: Inheritance

1. What is Inheritance?

Inheritance is a fundamental concept in object-oriented programming where one class (child or subclass) inherits the attributes and behaviors (methods) of another class (parent or superclass). It allows for code reuse and logical hierarchy.

2. Syntax of Inheritance

In Java, inheritance is implemented using the extends keyword:

class Animal {
  void eat() {
    System.out.println("This animal eats food.");
  }
}

class Dog extends Animal {
  void bark() {
    System.out.println("The dog barks.");
  }
}

3. Real-World Analogy

Imagine a generic Vehicle class. A Car and a Bike both are types of vehicles. They inherit general behavior like start() or stop() but also have unique methods.

5. Why Use Inheritance?

  • Code Reusability: Avoid rewriting code.
  • Logical Structure: Represent real-world relationships.
  • Maintainability: Easier to update and manage.

6. Types of Inheritance in Java

  • Single Inheritance: One child class inherits from one parent class.
  • Multilevel Inheritance: Class B inherits from A, and C inherits from B.
  • Hierarchical Inheritance: Multiple child classes inherit from a single parent.

7. Practical Examples

  • Employee Management System: Class Employee can be extended by Manager, Engineer, etc.
  • Game Characters: Character class can be extended into Warrior, Archer, Wizard.
  • E-Commerce Platform: Product can be a base class for Book, Electronics, etc.

8. The super Keyword

The super keyword is used to refer to the parent class. It can be used to call the parent's constructor or methods:

class Animal {
  void eat() {
    System.out.println("Animal eats");
  }
}

class Dog extends Animal {
  void eat() {
    super.eat();
    System.out.println("Dog eats bones");
  }
}

9. Fun Fact!

Java does not support multiple inheritance with classes to avoid the "diamond problem". However, it allows multiple inheritance through interfaces!

11. Summary

  • Inheritance lets a class inherit from another class.
  • Use extends to implement inheritance.
  • super keyword accesses parent class features.
  • Promotes code reuse and real-world modeling.
