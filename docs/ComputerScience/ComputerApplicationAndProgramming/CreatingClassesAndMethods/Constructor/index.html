Java Programming: Constructor Methods

Welcome! In this lesson, we will dive deep into Constructor Methods in Java. Constructors are a fundamental concept used when creating objects. This topic is especially important for building robust and reusable code in Java.

1. What is a Constructor?

A constructor in Java is a special method used to initialize objects. It is automatically called when a new object is created using the new keyword.

2. Constructor Syntax

class Dog {
  String name;

  // Constructor
  Dog(String dogName) {
    name = dogName;
  }
}

Here, the constructor Dog(String dogName) initializes the name attribute when a new Dog object is created.

3. Real-World Example: Bank Account

class BankAccount {
  String accountHolder;
  double balance;

  BankAccount(String name, double initialDeposit) {
    accountHolder = name;
    balance = initialDeposit;
  }

  void displayInfo() {
    System.out.println("Account Holder: " + accountHolder);
    System.out.println("Balance: $" + balance);
  }
}

5. Default Constructors

If no constructor is defined, Java provides a default no-argument constructor:

class Car {
  String brand;
}

Car myCar = new Car(); // uses the default constructor

6. Constructor Overloading

You can define multiple constructors with different parameter lists. This is called constructor overloading:

class Rectangle {
  int length, width;

  // Constructor 1
  Rectangle() {
    length = 0;
    width = 0;
  }

  // Constructor 2
  Rectangle(int l, int w) {
    length = l;
    width = w;
  }
}

7. Fun Fact

The concept of constructors in object-oriented programming comes from Simula, the first object-oriented language developed in the 1960s. Java adopted and enhanced these principles!

8. Practical Applications

  • Games: Initialize characters with health, position, and attributes using constructors.
  • Finance Apps: Set up bank accounts, credit cards, and transactions with default or custom values.
  • Mobile Apps: Load user profiles using constructors when the app launches.
