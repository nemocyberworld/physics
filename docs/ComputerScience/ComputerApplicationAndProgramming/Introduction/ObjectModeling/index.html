Object Oriented Development Themes

Introduction

Object-Oriented Programming (OOP) is a paradigm in software development that organizes software design around data, or objects, rather than functions and logic. It enables developers to build applications that are more modular, reusable, and easier to maintain.

Core Themes of Object-Oriented Development

1. Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class with actual values.

class Car {
  constructor(make, model) {
    this.make = make;
    this.model = model;
  }

  displayInfo() {
    return `Car: ${this.make} ${this.model}`;
  }
}

const myCar = new Car("Toyota", "Corolla");
console.log(myCar.displayInfo());

2. Encapsulation

Encapsulation hides the internal state of an object and only exposes safe parts of it. This protects the data.

class BankAccount {
  #balance;

  constructor(initialBalance) {
    this.#balance = initialBalance;
  }

  deposit(amount) {
    this.#balance += amount;
  }

  getBalance() {
    return this.#balance;
  }
}

const account = new BankAccount(100);
account.deposit(50);
console.log(account.getBalance());

3. Inheritance

Inheritance allows a class to acquire properties and methods of another class.

class Animal {
  speak() {
    return "Animal speaks";
  }
}

class Dog extends Animal {
  speak() {
    return "Dog barks";
  }
}

const pet = new Dog();
console.log(pet.speak());

4. Polymorphism

Polymorphism means many forms â€” the same function behaves differently based on the object.

function makeItSpeak(animal) {
  console.log(animal.speak());
}

makeItSpeak(new Animal());
makeItSpeak(new Dog());

Practical Example: User Login System

class User {
  constructor(username, password) {
    this.username = username;
    this.password = password;
  }

  login(inputPassword) {
    return this.password === inputPassword ? "Login successful" : "Login failed";
  }
}

const user1 = new User("admin", "1234");
console.log(user1.login("1234"));
