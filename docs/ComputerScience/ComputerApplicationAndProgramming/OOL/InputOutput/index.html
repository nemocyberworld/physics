Object-Oriented Programming in C++

Managing Console I/O Operations

Console Input/Output (I/O) operations form the backbone of interaction between a C++ program and the user. Understanding how to effectively manage console I/O is crucial for creating user-friendly applications.

1. Basic Console Output with cout

In C++, the cout object, part of the iostream library, is used to output data to the console. 

#include <iostream>
using namespace std;

int main() {
  cout << "Hello, world!" << endl;
  return 0;
}

Here, endl inserts a newline character and flushes the output buffer.

2. Console Input Using cin

The cin object is used to read input from the keyboard. 

#include <iostream>
using namespace std;

int main() {
  int age;
  cout << "Enter your age: ";
  cin >> age;
  cout << "You entered: " << age << endl;
  return 0;
}

3. Managing Input Errors

Sometimes users enter invalid input. We can check for errors and clear the input buffer. 

#include <iostream>
using namespace std;

int main() {
  int number;
  cout << "Enter a number: ";
  while (!(cin >> number)) {
    cout << "Invalid input. Please enter a valid number: ";
    cin.clear();               // Clear error flag
    cin.ignore(10000, '\n');   // Discard invalid input
  }
  cout << "You entered: " << number << endl;
  return 0;
}

4. Using cerr and clog

Besides cout, C++ provides:

  • cerr: Used for error messages (unbuffered output).
  • clog: Used for logging messages (buffered output).
#include <iostream>
using namespace std;

int main() {
  cerr << "Error: Invalid operation!" << endl;
  clog << "Log: Operation attempted." << endl;
  return 0;
}

5. Formatting Output with Manipulators

The iomanip library provides useful manipulators to format output:

  • setw(n): Set width of the next output.
  • setprecision(n): Set decimal precision for floating-point output.
  • fixed: Use fixed-point notation.
  • left and right: Align output.
#include <iomanip>
#include <iostream>
using namespace std;

int main() {
  cout << setw(10) << left << "Name"
       << setw(5) << right << "Age" << endl;
  cout << setw(10) << left << "Alice"
       << setw(5) << right << 30 << endl;
  cout << setw(10) << left << "Bob"
       << setw(5) << right << 25 << endl;

  cout << fixed << setprecision(2);
  cout << "Pi approx: " << 3.14159265 << endl;

  return 0;
}

6. Real-World Example: Simple Console Menu

Here is a simple interactive menu for a console application: 

#include <iostream>
using namespace std;

int main() {
  int choice;

  do {
    cout << "\nMenu:\n";
    cout << "1. Say Hello\n";
    cout << "2. Display Date\n";
    cout << "3. Exit\n";
    cout << "Enter your choice: ";
    cin >> choice;

    switch(choice) {
      case 1:
        cout << "Hello, User!" << endl;
        break;
      case 2:
        cout << "Today's date is May 15, 2025." << endl;
        break;
      case 3:
        cout << "Exiting program." << endl;
        break;
      default:
        cout << "Invalid choice. Try again." << endl;
    }
  } while(choice != 3);

  return 0;
}
