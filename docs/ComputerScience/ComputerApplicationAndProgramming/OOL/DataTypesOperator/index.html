Data Types and Operators in C++

1. Introduction

Understanding data types and operators is crucial for programming in C++. Data types define what kind of data a variable can store, and operators allow you to perform operations on these variables.

2. Data Types in C++

C++ provides several built-in data types that store different kinds of data.

int      // Integer numbers, e.g. 42, -3
float    // Floating-point numbers (decimals), e.g. 3.14
double   // Double precision floating-point, e.g. 3.141592
char     // Single character, e.g. 'A', 'z'
bool     // Boolean true or false

Example declaration:

int age = 30;
float price = 19.99;
char grade = 'A';
bool isRegistered = true;

3. Derived and User-defined Data Types

C++ also supports:

  • Arrays: Collections of data items of the same type.
  • Strings: Character arrays or std::string class.
  • Structures (struct): User-defined groupings of variables.
  • Classes: Blueprints for creating objects (more on this in OOP lessons).
struct Customer {
  int id;
  char gender;
  bool isMember;
  double balance;
};

Customer customer1 = {101, 'M', true, 250.75};

4. Operators in C++

Operators perform operations on variables and values.

4.1 Arithmetic Operators

Used for mathematical calculations:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
int x = 10;
int y = 3;
int sum = x + y;       // 13
int diff = x - y;      // 7
int product = x * y;   // 30
int quotient = x / y;  // 3 (integer division)
int remainder = x % y; // 1

4.2 Relational Operators

Compare values and return boolean:

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater or equal
  • <= Less or equal
int age = 20;
bool isAdult = (age >= 18);  // true
bool isChild = (age < 13);   // false
bool isTeen = (age >= 13 && age < 20); // false

4.3 Logical Operators

  • && Logical AND
  • || Logical OR
  • ! Logical NOT
bool hasID = true;
bool hasTicket = false;
bool canEnter = hasID && hasTicket;  // false
bool canTry = hasID || hasTicket;    // true
bool cannotEnter = !hasID;            // false

5. Operator Precedence and Associativity

Operators have rules that define the order in which they are evaluated.

For example, multiplication (*) has higher precedence than addition (+):

int result = 5 + 3 * 2;  // equals 11, not 16

Parentheses can be used to override precedence:

int result = (5 + 3) * 2; // equals 16

7. Fun Fact

Did you know? The bool data type in C++ was officially introduced in the 1998 C++ standard. Before that, programmers often used integers (0 and 1) to represent true and false!

8. Summary

  • Data Types: Specify the kind of data stored (int, float, char, bool, etc.).
  • Operators: Perform operations on data (arithmetic, relational, logical).
  • Operator precedence and associativity affect how expressions are evaluated.
  • Understanding these basics is key for effective C++ programming and building complex applications.
