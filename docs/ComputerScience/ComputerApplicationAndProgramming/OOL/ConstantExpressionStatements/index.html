Declaration and Constants, Expression and Statements in C++

1. Introduction

In C++, declarations, constants, expressions, and statements form the foundation of all programs. Understanding these concepts is essential for mastering C++ and object-oriented programming.

2. Declaration

A declaration tells the compiler about the existence and type of variables, functions, or objects.

Every variable or object must be declared before use.

// Declaring variables in C++
int age;             // declares an integer variable named age
double price;        // declares a double precision floating-point variable
char grade;          // declares a character variable
bool isPassed;       // declares a boolean variable

You can also declare multiple variables in one line:

int x, y, z;

Initialization vs Declaration

Initialization is assigning an initial value during declaration:

int age = 25;          // declared and initialized
double price = 10.99;
char grade = 'A';
bool isPassed = true;

3. Constants

Constants are variables whose value cannot be changed once initialized. They protect your program from accidental modification of important values.

How to declare constants in C++?

const data_type variableName = value;
const double PI = 3.14159;
const int MAX_USERS = 100;
const char NEWLINE = '\n';

Why use constants?

  • Make programs easier to read and maintain
  • Prevent accidental changes to critical values
  • Allow the compiler to optimize the code better

Fun Fact

The const keyword is your program’s way of saying “Don’t touch this value!” — making your code safer and less error-prone.

4. Expressions

An expression is a combination of variables, constants, operators, and function calls that are evaluated to produce a value.

Examples of expressions:

5 + 3
age * 2
price / 4.5
(x + y) > z

Operators used in expressions:

  • Arithmetic operators: +, -, *, /, %
  • Relational operators: ==, !=, >, <, >=, <=
  • Logical operators: &&, ||, !
int result = 5 + 3 * 2;      // result is 11 because * has higher precedence
bool isAdult = (age >= 18);
bool passed = (score > 50) && (attendance > 75);

5. Statements

A statement is a complete instruction that performs some action. It can be a declaration, an expression followed by a semicolon, or a control statement.

Examples:

int x = 10;
y = x + 5;
cout << "Hello, world!" << endl;

Types of statements:

  • Declaration statements: Declare variables or constants.
  • Expression statements: Perform calculations or call functions.
  • Control statements: if, else, for, while, switch.
int age = 20;                      // declaration statement
age = age + 1;                    // expression statement
if (age >= 18) {                  // control statement
  cout << "You are an adult." << endl;
} else {
  cout << "You are a minor." << endl;
}

6. Practical Applications and Examples

Understanding declarations, constants, expressions, and statements helps you in every part of programming, especially in Object-Oriented Programming (OOP) where you work with classes and objects.

Example: Using constants in a class

class Circle {
  public:
    static const double PI;       // constant declaration
    double radius;

    Circle(double r) {
      radius = r;
    }

    double area() {
      return PI * radius * radius;
    }
};

const double Circle::PI = 3.14159;  // constant definition outside class

int main() {
  Circle c(5);
  cout << "Area of circle: " << c.area() << endl;
  return 0;
}

7. Summary

  • Declaration: tells the compiler about variables, constants, or functions.
  • Constants: variables that cannot be changed after initialization; declared with const.
  • Expressions: combinations of variables, constants, and operators that evaluate to a value.
  • Statements: complete instructions ending with a semicolon.
  • These concepts are the building blocks of all C++ programs including Object-Oriented Programming.
