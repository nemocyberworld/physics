Lesson: Passing Parameters to Applets

1. What are Applet Parameters?

Parameters are values you can pass from an HTML page to a Java applet. They help customize behavior without altering code. Think of them like arguments passed to a function!

3. Reading Parameters in Java

In your applet class, use getParameter("parameterName"):

public void paint(Graphics g) {
  String user = getParameter("username");
  g.drawString("Hello, " + user + "!", 20, 20);
}
