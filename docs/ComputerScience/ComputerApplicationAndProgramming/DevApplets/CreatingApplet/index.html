Lesson: Creating Applets in Java

1. What is an Applet?

An applet is a Java program that can be embedded in a web page and run in a browser using the Java plugin. Although they are considered outdated, understanding applets is useful for learning how Java handles GUI components, threads, and lifecycle management.

2. The Basic Applet Structure

To create an applet, you need to extend the Applet class (or JApplet for Swing-based applets). Here's a simple example:

import java.applet.Applet;
import java.awt.Graphics;

public class MyApplet extends Applet {
  public void paint(Graphics g) {
    g.drawString("Welcome to Java Applet!", 20, 20);
  }
}

4. Lifecycle of an Applet

The applet lifecycle includes four main methods:

  • init(): Initialization logic
  • start(): Start or resume execution
  • paint(): Redraw the GUI
  • stop() and destroy(): Cleanup resources

5. Real-World Examples of Applets

  • Interactive physics simulations for education (pendulums, optics)
  • Simple online games (chess, puzzles)
  • Graphing calculators and plotting tools
  • Early e-commerce product configurators

6. Adding User Interaction

You can capture mouse and keyboard input using Java's event handling. Here's an example of handling a mouse click:

import java.applet.*;
import java.awt.*;
import java.awt.event.*;

public class ClickApplet extends Applet implements MouseListener {
  int x = 0, y = 0;

  public void init() {
    addMouseListener(this);
  }

  public void paint(Graphics g) {
    g.drawString("You clicked at: (" + x + ", " + y + ")", x, y);
  }

  public void mouseClicked(MouseEvent e) {
    x = e.getX();
    y = e.getY();
    repaint();
  }

  // Unused methods
  public void mouseEntered(MouseEvent e) {}
  public void mouseExited(MouseEvent e) {}
  public void mousePressed(MouseEvent e) {}
  public void mouseReleased(MouseEvent e) {}
}

7. Common Mistakes When Creating Applets

  • Forgetting to override paint()
  • Not compiling to .class file before embedding
  • Misplacing the applet HTML in unsupported browsers

9. Modern Alternatives

Applets are now largely replaced by:

  • JavaFX: Modern GUI applications
  • Web technologies: HTML5, CSS3, JavaScript
  • Electron and Flutter: Cross-platform desktop and mobile apps

10. Summary

  • Applets are small Java programs embedded in web pages.
  • They follow a strict lifecycle managed by the browser or applet viewer.
  • They serve as a great way to learn Java GUI basics.
  • Modern Java applications use JavaFX or other frameworks.
