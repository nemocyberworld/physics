Lesson: Including Applets on a Web Page

1. Introduction

Including a Java Applet in a web page used to be a common way to bring interactivity and advanced graphics into websites before JavaScript and HTML5 took over. While applets are now largely deprecated, learning how to include them can help you understand legacy systems and how early interactive content was delivered online.

2. Prerequisites

  • Java Development Kit (JDK) installed
  • A compiled .class file of your applet
  • Basic understanding of HTML

4. Example

Let's say we have an applet class that displays a greeting. Here's the Java code:

import java.applet.Applet;
import java.awt.Graphics;

public class GreetingApplet extends Applet {
  public void paint(Graphics g) {
    g.drawString("Hello from the Applet!", 20, 20);
  }
}

6. Modern Compatibility

  
Most modern browsers no longer support applets due to security concerns and lack of plugin support. Instead, tools like Java Web Start, JavaFX, or converting logic to JavaScript using tools like GWT (Google Web Toolkit) are recommended.

Did you know? The <applet> tag was one of the first ways to bring mini-programs into the web. This helped Java become the most popular language of the early 2000s!
