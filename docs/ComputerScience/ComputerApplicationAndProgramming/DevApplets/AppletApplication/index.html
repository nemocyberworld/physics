Lesson: Applets and Applications in Java

1. Introduction

Java supports two types of programs: Applications and Applets. Applications are stand-alone programs that run independently, while Applets are small programs that run inside a web browser or an applet viewer.

3. Java Applets

Applets are classes that extend the Applet class or JApplet (Swing version). They override lifecycle methods like init(), start(), stop(), and destroy().

import java.applet.Applet;
import java.awt.Graphics;

public class HelloApplet extends Applet {
  public void paint(Graphics g) {
    g.drawString("Hello Applet!", 20, 20);
  }
}

4. Applet Lifecycle

Applets go through a sequence of states:

  • init(): Called once when the applet is loaded.
  • start(): Called after init or when restarting.
  • paint(): Called to redraw the interface.
  • stop(): Called when the applet is paused.
  • destroy(): Called when the applet is terminated.

5. Real-World Applications

Java applets were once popular for web-based games, interactive simulations, and visual demonstrations in education. Though modern browsers have dropped support, they remain a great learning tool.

  • Educational tools: Simulate physics or chemistry experiments.
  • Financial dashboards: Display stock tickers and graphs (historically).
  • Data visualization: Graph plotting tools embedded in web pages.

6. Applications in Practice

Today, applet ideas live on in desktop and mobile applications:

  • Interactive learning apps (e.g., Khan Academy)
  • Weather widgets
  • Mini-games and puzzles

7. Fun Fact

Did you know? The original version of the game Minecraft was written in Java and had elements similar to applets in how it was embedded in browsers!

8. Security Considerations

Due to security risks, modern browsers block applets by default. Java Web Start or JavaFX are now used instead. Always avoid running untrusted applets.

9. Summary

  • Applications run on JVM with a main method.
  • Applets run in browsers and have a lifecycle.
  • Modern use of applets is mostly educational or legacy support.
